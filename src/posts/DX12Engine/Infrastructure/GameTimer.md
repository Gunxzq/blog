# GameTimer（游戏计时器）

## 1. 定位与职责

GameTimer 提供高精度计时能力，是帧循环的"心跳"来源。

- **上游依赖**：无（纯工具类）
- **下游服务**：FrameDriver（计算 DeltaTime）、动画系统、物理模拟

### 核心职责

| 职责 | 说明 |
|:----|:-----|
| **高精度计时** | 基于 QueryPerformanceCounter 提供微秒级计时 |
| **DeltaTime 计算** | 每帧帧间隔时间，用于逻辑更新 |
| **帧率统计** | 实时 FPS 计算 |
| **时间缩放** | 支持游戏时间缩放（慢动作/加速） |

---

## 2. 生命周期

```
创建 → Start() → 每帧 Tick() → ... → Stop() → 销毁

关键事件：
- 暂停/恢复：暂停时 DeltaTime = 0，恢复后继续
- 时间缩放：缩放因子影响 DeltaTime 但不影响物理计时
```

---

## 3. 设计原则

| 原则 | 说明 |
|:----|:-----|
| **单一数据源** | 所有时间相关模块共用同一个 GameTimer 实例 |
| **非阻塞** | 计时操作不涉及 I/O 或同步，零开销 |
| **时间缩放隔离** | 逻辑时间缩放不影响物理/网络同步的绝对时间 |
