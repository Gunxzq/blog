# VisibleRaycaster（可见集射线检测）

## 1. 定位与职责

VisibleRaycaster 提供 **可见集内的射线检测** 能力，用于拾取、交互等场景。

- **上游依赖**：ECS::Registry（实体查询）
- **下游服务**：Editor 拾取、Game 交互

### 核心职责

| 职责 | 说明 |
|:----|:-----|
| **射线检测** | 在可见实体集中检测射线交点 |
| **结果缓存** | 每帧检测结果存入 GameContext（raycastResult） |
| **与 Culling 协作** | 仅在可见实体中检测，减少计算量 |

---

## 2. 数据流

```
输入：射线（起点 + 方向）
  → 查询可见实体集
  → 对每个实体的包围盒/网格进行相交测试
  → 返回最近的交点（实体 ID + 位置 + 法线）
```

---

## 3. 设计原则

| 原则 | 说明 |
|:----|:-----|
| **可见集过滤** | 仅检测已标记为可见的实体，排除不可见实体 |
| **结果共享** | 检测结果存入 GameContext，供多个消费者使用 |
| **精确检测可选** | 先包围盒粗筛，再按需精确网格检测 |
