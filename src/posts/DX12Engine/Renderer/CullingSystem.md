# CullingSystem（可见性剔除系统）

## 1. 定位与职责

CullingSystem 负责 **可见性剔除**，减少不必要的渲染调用。

- **上游依赖**：ECS::Registry（实体查询）、CameraManager（视锥体）
- **下游服务**：渲染 System（提供可见实体列表）

### 核心职责

| 职责 | 说明 |
|:----|:-----|
| **视锥体裁剪** | 剔除不在摄像机视锥体内的实体 |
| **可见集标记** | 标记可见/不可见实体，供渲染 System 使用 |
| **遮挡查询** | 可选：硬件遮挡查询（未来扩展） |

---

## 2. 执行阶段

CullingSystem 在 **PostCulling** 阶段执行，位于 EarlyUpdate/Update/LateUpdate 之后、SceneDataUpload 之前。

---

## 3. 设计原则

| 原则 | 说明 |
|:----|:-----|
| **非破坏性** | 剔除仅标记可见性，不修改实体或组件数据 |
| **可扩展** | 支持多种剔除算法（视锥体、遮挡、距离） |
| **与 LOD 协作** | LODSystem 在剔除后使用可见集进行 LOD 选择 |
