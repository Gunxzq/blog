# LODSystem（层次细节系统）

## 1. 定位与职责

LODSystem 根据实体与摄像机的距离，自动选择合适精度的网格模型。

- **上游依赖**：CameraManager（视点位置）、GeometryResourceManager（网格数据）
- **下游服务**：渲染 System（提供 LOD 级别信息）

### 核心职责

| 职责 | 说明 |
|:----|:-----|
| **LOD 级别选择** | 根据距离切换 LOD 0/1/2/3 |
| **LOD 配置** | 每个级别对应的距离阈值、屏幕占比 |
| **与 Culling 协作** | 在剔除后执行，仅对可见实体计算 LOD |

---

## 2. LOD 级别策略

| LOD 级别 | 用途 | 典型距离 |
|:---------|:-----|:---------|
| LOD 0 | 最高精度（完整网格） | 近距离 |
| LOD 1 | 中等精度（简化网格） | 中距离 |
| LOD 2 | 低精度（极简网格） | 远距离 |
| LOD 3 | 仅包围盒/不渲染 | 极远距离 |

---

## 3. 设计原则

| 原则 | 说明 |
|:----|:-----|
| **配置驱动** | 距离阈值、级别数通过 LODConfig 配置 |
| **与 Culling 解耦** | 剔除和 LOD 是独立的两个阶段，但共享可见集 |
| **实体级别** | LOD 是实体级别的，每个实体独立的 LOD 状态 |
