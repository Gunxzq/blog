# SkeletonManager（骨骼管理器）

## 1. 定位与职责

SkeletonManager 管理 **骨骼动画数据**，包括骨骼层次结构、绑定姿势和蒙皮数据。

- **上游依赖**：无（纯 CPU 数据管理）
- **下游服务**：蒙皮渲染 System（获取骨骼变换矩阵）

### 核心职责

| 职责 | 说明 |
|:----|:-----|
| **骨骼注册** | 注册骨骼数据，返回骨骼句柄 |
| **骨骼查询** | 按句柄查询骨骼层次结构和变换数据 |
| **引用计数** | 管理骨骼数据的生命周期 |

---

## 2. 设计原则

| 原则 | 说明 |
|:----|:-----|
| **与动画系统分离** | 骨骼管理器只管理静态骨骼数据，动画状态由 ECS 动画组件管理 |
| **GPU 数据上传** | 骨骼变换矩阵由蒙皮渲染 System 负责上传到 GPU |
